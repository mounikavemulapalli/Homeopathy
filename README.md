# 🏥 Homeopathy Case Management System

This is a full-stack web application designed to manage homeopathic patient cases, generate summaries, and provide intelligent support using AI. Built with React, Node.js, and the Gemini API.

---

## 🚀 Features

- ✍️ Add, Edit, and View Patient Cases
- 📋 Repertorization and Summary Generation using AI
- 📄 Case-wise PDF Export
- 🧪 Support for Lab Reports, Skin/Face Analysis
- 🌐 Responsive UI
- 🔐 Secure form-based access

---

## 🛠️ Tech Stack

| Frontend        | Backend         | AI/ML            | Database      |
|-----------------|------------------|------------------|---------------|
| React.js        | Node.js + Express| Gemini API (LLM) | MongoDB (Mongoose) |
